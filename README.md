# 🗂️ Kanban Board Project

A dynamic Kanban board web application built using HTML, CSS, and JavaScript. It helps users manage tasks efficiently with drag-and-drop functionality and persistent storage.

---

## ✨ Features
- ✅ Drag and Drop tasks
- ✅ Add new tasks
- ✅ Delete tasks
- ✅ Data stored using localStorage
- ✅ Responsive UI
  
---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
